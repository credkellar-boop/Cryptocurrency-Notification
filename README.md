🚀 Noti_Fy_Crypto_Bot: High-Vibration Notification HubAn automated system for real-time cryptocurrency transaction alerts. This hub tracks, notifies, and manages all incoming and outgoing on-chain activity with advanced biometric validation and manual override capabilities. Key Features⚡ Real-Time Tracking: Instant notifications for all incoming/outgoing transactions.🛡️ Biometric Security: Advanced owner_biometrics integration for secure session handling.🎮 Manual Override: Interactive Telegram buttons to approve, halt, or modify transactions on the fly.🔗 Multi-Chain Support: Seamless connection to Ethereum, Monad, and other EVM-compatible chains via blockchain_listener.py.🩺 High-Vibration Heartbeat: GitHub Actions-powered health checks to ensure 99.9% uptime for HFT nodes.📂 Project StructurePlaintext📁 .github/workflows    <-- CI/CD & Heartbeat monitoring
📁 owner_biometrics     <-- Secure identity verification
📄 blockchain_listener  <-- Core chain monitoring logic
📄 bot.py              <-- Telegram Interface & Logic
📄 main.py             <-- Application Entry Point
