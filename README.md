# Cryptocurrency-Notification Hub

Automated system for real-time cryptocurrency transaction alerts. Tracks and notifies all incoming (receive) and outgoing (send) on-chain activity with manual **Approve/Deny** controls via Telegram.

## Features
- **Real-time Monitoring:** Connects to Ethereum/EVM chains via Web3.
- **Manual Override:** Interactive buttons to approve or halt transactions.
- **Secure:** Environment-based credential management.

## Setup
1. Clone the repo: `git clone https://github.com/credkellar-boop/Cryptocurrency-Notification.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Configure your `.env` file with your Bot Token and Provider URL.
4. Run the bot: `python bot.py`
