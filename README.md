🚀 **Noti_Fy_Crypto_Bot**: A High-Vibration Notification Hub and automated system for real-time cryptocurrency transaction alerts. This hub tracks, notifies, and manages all incoming (receive) and outgoing (send) on-chain activity with advanced biometric validation and manual override capabilities.

## ⚡ Key Features
*   **Real-Time Tracking**: Instant notifications for all incoming/outgoing transactions.
*   **Biometric Security**: Advanced `owner_biometrics` integration for secure session handling.
*   **Manual Override**: Interactive Telegram buttons to approve, halt, or modify transactions on the fly.
*   **Multi-Chain Support**: Seamless connection to Ethereum, Monad, and other EVM-compatible chains via `blockchain_listener.py`.
*   **High-Vibration Heartbeat**: GitHub Actions-powered health checks to ensure 99.9% uptime for HFT nodes.

## 🛠 Project Infrastructure
*   **Core Logic**: `bot.py` and `blockchain_listener.py`
*   **Security Layer**: `owner_biometrics` and `security_wipe.py`
*   **Deployment**: Ready for containerization via `Dockerfile` and `Procfile`.
